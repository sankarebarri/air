# Cookbook

A handy Air-themed list of recipes for common web app tasks.

This section contains short recipes. See the topic pages for details:

- **[app.page decorator](http://feldroy.github.io/air/learn/cookbook/page-decorator/index.md)** - how to use the `@app.page` decorator to quickly create web pages
- **[Authentication](http://feldroy.github.io/air/learn/cookbook/authentication/index.md)** - how to handle login and authentication in your application
- **[Bigger applications](http://feldroy.github.io/air/learn/cookbook/bigger-applications/index.md)** - how to structure larger applications
- **[Charts](http://feldroy.github.io/air/learn/cookbook/charts/index.md)** - Examples of using charts in Air
- **[Minimal app](http://feldroy.github.io/air/learn/cookbook/minimal/index.md)** - The smallest possible Air app
- **[Forms & validation](http://feldroy.github.io/air/learn/cookbook/forms/index.md)** - how to create forms and validate user inputs
- **[Static files](http://feldroy.github.io/air/learn/cookbook/static/index.md)** - how to serve static files like CSS and JavaScript

If you'd like to contribute a recipe, open a PR with a new file under `docs/cookbook`.
